Data Policy

Data Handling

  • No PHI/PII is stored or transmitted outside the host system.
  • Inputs are processed in-memory for inference and discarded.
  • Access logs exclude patient-identifying information.

Security

  • Local-only architecture by default; no third-party calls.
  • Recommend HTTPS and network isolation in production.
  • Optional authentication and audit trails can be added.
