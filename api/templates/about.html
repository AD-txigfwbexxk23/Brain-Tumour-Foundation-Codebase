About

This system offers decision support for neuro-oncology by estimating survival rate and recommending a treatment strategy based on key clinical variables. It is designed for professional use and to complement—not replace—clinical judgment.

What the system does

  • Processes 7 structured inputs (4 numeric, 3 categorical)
  • Predicts survival rate (regression) and treatment (classification)
  • Provides class probabilities to show recommendation confidence

Clinical positioning

  • Use as a second-opinion tool alongside multidisciplinary review
  • Supports rapid triage and documentation of rationale
  • Configurable to local pathways and risk thresholds
Model
Multi-output NN
Framework
PyTorch + FastAPI
Latency
~10–50 ms (CPU)
Data residency
Local only

Privacy & Security

  • No data leaves the server; no external APIs
  • No persistence of patient identifiers
  • Stateless inference; logs contain no PHI by default

Usage Guidance

  • Confirm inputs match clinical notes and imaging
  • Use class probabilities to understand uncertainty
  • Document rationale when deviating from suggestion
